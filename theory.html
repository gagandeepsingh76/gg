Phase Shift Oscillator using Operational Amplifier

Introduction

A phase shift oscillator using an operational amplifier (Op-amp) is a variation of the traditional phase shift oscillator where an op-amp replaces a transistor or other amplifying device. This design is commonly used to generate stable sine waves for audio signal generation, function generators, and test equipment. The phase shift circuit consists of a three-stage RC network that creates a 180˚ phase difference between input voltage (Vin) and output voltage (Vout), as shown in Figure 1.

Fig. 1: Phase shift RC network with three RC combinations.

When an AC voltage is applied to an RC combination like C1 and R1, the current through the capacitor leads the current through the resistor by 90˚. Consequently, the AC voltage experiences a phase lag θ as it passes through. Equation (1) gives the value of θ:

tan θ = VC1 / VR1 = XC1 / R1

where XC1 is the capacitive reactance of C1, and R1 is the resistance. Generalizing this:

XC = 1 / (2πfC)

Therefore, equation (1) becomes:

tan θ = 1 / (2πfR1C1)

θ = tan-1(1 / 2πR1C1)

This shows θ can be tuned by choosing appropriate R1C1 values. If each stage (R1C1, R2C2, R3C3) gives a 60˚ shift, the total shift is 180˚, forming a complete phase shift network.

Assuming R1 = R2 = R3 = R and C1 = C2 = C3 = C, the oscillation frequency is:

f = 1 / (2π√6RC)

β = Vout / Vin = 1 / 29

At frequency f, the input and output are 180° out of phase. To complete oscillation, an active component is needed to provide an additional 180° phase shift and gain of 29.

Based on this principle, Figure 2(a) shows a phase shift circuit using an op-amp.

Figure 2(a): Op-Amp Based Phase Shift Oscillator

The phase shift network includes three RC combinations and a feedback resistor at point B, contributing to the negative feedback:

Ainv = -Rf / R1

Rf is adjusted so the amplifier gain becomes 29, offsetting β.

Figure 2(b): Transistor-Based Phase Shift Oscillator

Similar results are achieved using a CE amplifier with the phase shift network, where the collector connects to the input (A), and the base connects to the output. The amplifier must support a gain above 29.

Working Principle:

The oscillator uses a phase shift RC network and an op-amp in feedback to sustain oscillation. The op-amp provides the necessary inversion and gain.

Components and Configuration:

  1. Op-Amp: Provides the gain and inversion needed.
  2. RC Network:
    • Three RC stages in series. Each provides ~60˚ shift, totaling 180˚.
    • Combined with the amplifier’s 180˚, total phase shift is 360˚.
  3. Feedback Loop: Output is looped back to input through the RC network.
  4. Gain Requirement: Op-amp gain ≥ 3 is needed to compensate signal attenuation.

Condition for Oscillation (Barkhausen Criterion):

Oscillation is sustained when:

  • Total phase shift = 360˚ (RC: 180˚ + Amp: 180˚)
  • Loop gain ≥ 1 (amplifier gain ≥ 3)

Frequency of Oscillation:

Depends on R and C in the RC network:

f = 1 / (2π√6RC)

  • f = frequency
  • R = resistance
  • C = capacitance

Advantages of Phase Shift Oscillator Using Op-Amp:

  1. Simple Design: Easier than Colpitts or Hartley.
  2. Pure Sine Wave: Produces clean sine output.
  3. Stable: Op-amps offer high stability.
  4. Low Cost: Easy and affordable to build.

Disadvantages:

  1. Component Sensitivity: Requires precise RC values.
  2. Limited Frequency: Bound by op-amp and RC limits.
  3. Gain Requirements: Needs sufficient op-amp gain.

Applications:

  1. Signal Generators – For test/audio tones.
  2. Audio Equipment – Tone generation and testing.
  3. Function Generators – Produces sine waveforms.
  4. Test Equipment – Reliable sine wave sources.
